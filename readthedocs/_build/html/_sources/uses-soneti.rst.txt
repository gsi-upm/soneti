Crawling
--------

This video shows you the basic behaviour of **GSICrawler**. GSICrawler provides an interface to crawl a number of web sites, based on Scrapy_. The results of Scrapy are returned as a JSON-LD (JSONLD_, JSON for Linking Data).  This file  can be processed and stored later by the Analysis modules. In this way, the scrapped information is mapped onto a number of ontologies and can be linked and queried. If you need to scrape a new web site, a new spider should be developed in Python. 

.. _Scrapy: https://scrapy.org/

.. _JSONLD: https://json-ld.org/

Sentiment analysis
------------------
The tool **Senpy** has been used for sentiment and emotion analysis from social networks. **Senpy** provides semantic interoperability across different analyzers, so you do not need to change your code to use a new analyzer. The main advantages of Senpy (Senpy_) are that (i) it provides out-of-the-box facilities for transforming the analysis into linked data and (ii) it follows a plugin-in architecture which makes it easy to extend.


.. _Senpy: http://senpy.readthedocs.io
